# 一、功能介绍

## 简介

1. 用户在代码中添加RM打印相关注释语句
2. 脚本自动查找注释语句并解析为字典
3. 脚本依据字典内容生成RM开关文件和Define宏文件
4. 脚本自动将打印宏添加至源代码
